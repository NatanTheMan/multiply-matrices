# Multiply Matrices

Type two matrices and the program will multiply them.

## Requirements

- PHP
- Composer

Instructions to install them: [PHP](https://www.php.net/downloads.php), [Composer](https://getcomposer.org/download/)

## Run

```bash
git clone https://github.com/NatanTheMan/multiply-matrices.git
```

```bash
cd multiply-matrices
```

```bash
composer run start
```
